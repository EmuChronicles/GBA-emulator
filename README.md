# GBA-emulator

## Descrição
Desenvolvimento de um emulador funcional do console portátil Game Boy Advance (GBA), feito em dupla, com foco em aprendizado e portfólio nas áreas de arquitetura de computadores, programação de baixo nível e otimização de performance.
