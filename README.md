# GBA-emulator

## Sobre o projeto
O objetivo deste projeto é desenvolver, em dupla, um emulador funcional do console portátil Game Boy Advance (GBA). O proposito principal é servir como projeto de portfolio aprendizado em áreas fundamentais da computação, como arquitetura de computadores, programação de baixo nível e otimização de performance.
O projeto ainda está em etapa inicial/embrionária, por aqui será feita do a documentação e o versionamento do projeto.
